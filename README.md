# 기상룩
### 기상⏰후에 기상☀️에 맞는 옷차림👕을 알 수 있는 날씨앱,</br>
### '기상룩(WeatherLook)'입니다.
**매일 아침 일어나는 '기상'** 과 **날씨 '기상'** 의 중의적 의미를 담고 있으며,</br>
날씨에 따른 옷차림 가이드를 통해 매일 아침 '오늘 뭐 입지?'에 대한 고민을 도와주는 서비스입니다.
